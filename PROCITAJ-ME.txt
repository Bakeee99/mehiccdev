  mehiccdev — POPRAVAK: sadržaj nestaje pri promjeni jezika/teme
═══════════════════════════════════════════════════════════

PROBLEM:
  Kad promijeniš jezik ili temu, neke sekcije postanu nevidljive
  dok ne reloadaš stranicu.

UZROK:
  Sekcije se animiraju "na scroll" (initial="hidden" + whileInView).
  Pri promjeni jezika/teme sve se ponovo iscrta, elementi se vrate
  na "hidden" (nevidljivo), ali se animacija ne okine ponovo jer si
  već skrolovao tu — pa ostanu nevidljivi do reloada.

POPRAVAK (u ovom zip-u):
  Svaka sekcija sad pamti da je već prikazana (stanje "seen").
  Nakon prvog prikaza, na svako sljedeće iscrtavanje element ostaje
  vidljiv (ne sakriva se ponovo). Prva animacija na scroll i dalje
  radi normalno.

ŠTA JE U ZIP-U (8 ispravljenih sekcija):
  components/sections/About.tsx
  components/sections/Contact.tsx
  components/sections/Footer.tsx
  components/sections/Portfolio.tsx
  components/sections/Pricing.tsx
  components/sections/SaasTeaser.tsx
  components/sections/Satisfaction.tsx
  components/sections/Services.tsx

KAKO UBACITI:
  1. Raspakuj U KORIJEN projekta (gdje su folderi components, lib, app).
  2. Na pitanje "Replace / Zamijeni?" → klikni DA za sve.
  3. Pokreni:  npm run dev
     Testiraj: skroluj dole, pa mijenjaj jezik/temu — ništa ne smije
     nestati.
  4. Ako je OK:
        git add .
        git commit -m "Fix: sadrzaj nestaje pri promjeni jezika/teme"
        git push

NAPOMENA:
  • Provjereno: cijeli projekt se uspješno builda (npm run build).
  • Results i Value sekcije već rade ispravno (drugi pristup) — nisu
    mijenjane.
